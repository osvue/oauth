# 三层架构思想, 测试驱动编程
**注:项目为回归项目,并不作为长期支持版本**
 极限编程
 * 注册中心 Nacos
 * 数据访问 Mybatis Plus
 * 微服务生态 spring cloud G & spring cloud Alibaba
 * 服务通信 Apache dubbo Rpc
 * 安全框架 Spring Security OAuth2.0  (自定义Client 查询 & redis 存储)
 * 前台使用 vue-admin-template 模板 实现简单SSO 无状态单点登录
 
 **provider** 提供数据访问
 
 **business** 业务逻辑处理
